<p>
How To Run:
</p>
<pre>  gem install sinatra
  cd challenge
  rake server</pre>
<p>
View the app here:
</p>
  <pre>  http://localhost:9292</pre>

<p>
You can view the dummy json here:
</p>
  <pre>  http://localhost:9292/chat-events</pre>

<p>You can run the jasmine tests by opening this file in your browser:</p>
  <pre>  challenge/tests/SpecRunner.html</pre>
